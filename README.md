# Games Master

A collection of data collection scripts in python.

## Setup

```sh
$ pip install -r src/requirements.txt
```

Create the config.json file.

```sh
$ touch static/config.json
```

```json
{
    "mobyGamesApiKey": "...",
    "giantBombApiKey": "...",
    "igdbClientId": "...",
    "igdbClientSecret": "...",
    "steamWebApiKey": "...",
    "itchioApiKey": "..."
}
```
